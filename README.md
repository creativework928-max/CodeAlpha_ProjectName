### 🔹 Task 1: Iris Flower Classification

* Built a classification model using the Iris dataset.
* Predicted flower species (*Setosa, Versicolor, Virginica*) based on input features.
* Applied machine learning algorithms using Scikit-learn.
* Evaluated model performance using accuracy metrics.

📊 **Key Concepts:**

* Classification
* Model training & testing
* Accuracy evaluation

* Trend analysis
* Economic data interpretation

* Evaluated model performance using regression metrics.

* Model evaluation
